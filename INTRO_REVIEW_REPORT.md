# Introduction Review Report - All 70 Topics

## Review Criteria
A good IELTS introduction should:
1. Stay GENERAL - not give specific reasons/effects/examples that belong in body
2. Paraphrase the topic (not copy it)
3. Present both views (for discuss-both-views questions)
4. State opinion clearly
5. Be 2-3 sentences (not too long)
